milvasic tools

A collection of single-file CLI tools for developer workflows.

  • motd

    A single-file Bash script that displays a system information dashboard at login.

    GitHub →
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/milvasic/motd/refs/heads/main/install.sh)"
  • mcli

    A single-file Bash CLI for managing Docker Compose services.

    GitHub →
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/milvasic/mcli/refs/heads/main/install.sh)"
  • tinit

    A single-file Bash CLI that sets up a Debian/Ubuntu terminal environment from scratch.

    GitHub →
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/milvasic/tinit/refs/heads/main/install.sh)"
  • distill

    A single-file Bash CLI that generates a self-contained install.sh for any CLI tool.

    GitHub →
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/milvasic/distill/refs/heads/main/install.sh)"
